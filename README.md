# Landing Page — Портфолио (Синергия)

Адаптивная посадочная страница (Landing), реализованная с использованием HTML5, CSS3 (Flexbox, Grid, медиа‑запросы). Проект подготовлен как часть портфолио в рамках обучения на фронтенд‑разработчика.

## Особенности реализации

- **Адаптивность**: полная поддержка мобильных устройств (медиа‑запросы, относительные единицы, `box-sizing: border-box`).
- **Сетки**: CSS Grid для разбивки секций, Flexbox для навигации и карточек.
- **Производительность**: минимальный CSS/JS, оптимизация изображений, семантическая разметка.

## Установка и запуск

Проект не требует сборки. Для запуска достаточно открыть `index.html` в браузере.

## Использование

Навигация переключается с горизонтальной на вертикальную при ширине экрана < 768px.
Блоки секций выстроены с помощью CSS Grid.

CSS3 (Flexbox, Grid, @keyframes, media queries)
